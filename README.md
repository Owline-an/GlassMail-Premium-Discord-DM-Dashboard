## 📌 Project Overview

**Discord DM Ticket Dashboard** is a powerful **centralized management platform** designed to transform Discord Direct Messages (DMs) into a **professional ticketing system**.

Instead of handling user messages inside Discord's crowded interface, this dashboard provides a **modern, customer-support-style UI** where you can:

* Manage conversations
* Reply to users instantly
* View user avatars and information
* Handle multiple servers from one place
* Maintain structured and professional support workflows

This project bridges the gap between **Discord's basic DM system** and the **advanced needs of large communities**.

---

## ✨ Key Features

# 💬 Professional Chat Suite

A fully featured messaging environment designed for professional support workflows.

### 📎 Full Media Support

Supports multiple content types directly inside conversations:

* 🖼 Images
* 🎥 Videos
* 📁 Files
* 🎭 Stickers

Everything appears inside the dashboard exactly as expected.

---

### 🎙 Voice Messages Support

Record voice messages **directly from your browser**, then send them to Discord where they appear as **professional audio players**.

**Supported Format:**

* `Ogg / Opus`

---

### 🔍 Smart Search System

Quickly locate conversations using:

* Username
* User ID
* Ticket name

Ideal for large-scale servers handling many users.

---

# 🔄 Smart Sync & Recovery System

Never miss a message again.

### 🔁 Offline Message Recovery

When the bot restarts after downtime:

* Automatically scans missed DM channels
* Retrieves messages sent during downtime
* Displays them as **new notifications**

---

### 📜 Load Message History

Dedicated feature to:

* Load archived messages
* Save conversation history locally
* Maintain a complete ticket timeline

---

# 🔔 WhatsApp-Style Notification System

Stay updated in real time with smart alerts.

### 🔴 Unread Message Counters

Red notification badges appear on:

* Servers
* Tickets

Showing **accurate unread message counts**.

---

### ⬆ Auto Sorting System

When a new message arrives:

* The ticket automatically jumps to the top
* Active conversations remain visible

---

### 🔊 Sound Notifications

Receive an audible alert when:

* A new message arrives
* A customer responds

Ensures **zero missed messages**.

---

# ⚙ Settings & User Interface

Premium-level customization and design.

### 🎚 Audio Settings Panel

Professional modal allowing:

* Volume adjustment
* Sound enable/disable
* Local browser saving of preferences

---

### 🎨 Premium UI Design

Modern interface built using:

* **Glassmorphism Design**

Delivering a **luxury user experience**.

## 🧠 How It Works

This project combines modern backend technologies with real-time communication.

---

### 🧩 Backend Core

Built using:

* **Node.js** — Main runtime
* **Discord.js** — Discord API interaction
* **Express.js** — Web server handling dashboard requests

---

### ⚡ Real-Time Communication

Powered by:

* **Socket.io**

Messages travel between:

Discord ⇄ Bot ⇄ Dashboard

in **milliseconds**, without page refresh.

---

### 🗄 Database System

Uses:

**SQLite** — Fast and lightweight local database.

Stores:

* Tickets
* Messages
* Archived data
* Read status (`isRead`)

Ensuring **data reliability** and persistence.

---

### 🔗 Data Integration System

The system merges:

* Discord user data
* Local database records

To display:

* Real usernames
* Profile pictures
* Ticket history

Creating a **human-friendly support experience**.

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Owline-an/your-repository-name.git
cd your-repository-name
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Configure Environment

Create:

```
.env
```

Add:

```env
DISCORD_TOKEN=your_bot_token
CLIENT_ID=your_client_id
```

---

### 4️⃣ Start the Bot

```bash
node index.js
```

---

### 5️⃣ Open Dashboard

Visit:

```
http://localhost:3000
```
